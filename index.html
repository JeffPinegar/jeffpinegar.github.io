Data Science Salaries

Using data from Kaggle, Salaries for those working in Data Science are examined. The data is a convenience sample from 2020-2022. So while the data is real, it may not represent any specific situation. We examine the data by Location, Job Title or position, employer size, and experience level. A deeper dive into Machine Learning is also undertaken. The full data set with the teams' embellishments is included on the data tab.
